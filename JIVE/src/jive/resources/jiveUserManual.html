Java Image Viewer and Editor (JIVE)

User Manual


1.   Getting Started

1.1.   Opening an Image File

To open an image file, select the menu button then select 'Open File', or press Ctrl+O. The menu button is the only button that is enabled when no image is loaded. JIVE supports .jpg, .gif, .bmp, and .png files. Attempting to open any other file type will result in an error message.

1.2.   Resizing the Window

Window resizing is handled by your operating system, including snap resizing. If an image is loaded it will be resized as necessary to fit into the viewer. The aspect ratio will always be preserved.

1.3.   The Photo Reel

After an image is loaded, other image files in the same directory are loaded to the photo reel. The photo reel bypasses the file chooser and loads the next or previous image from the directory into JIVE. Images are loaded in alphabetical order. The previous or next image can be loaded by pressing Ctrl+Left or Ctrl+Right, respectively. Or, by using the following buttons:

The buttons are only displayed when the mouse is hovered over them. The relative location is shown below:

If there are no next or previous pictures, the corresponding button will not be displayed and the keyboard shortcut will be disabled. If the current image has unsaved changes, you will be prompted to save the changes before the new image is loaded.

1.4.   Keyboard Shortcuts

Keyboard shortcuts for certain functions are noted in the relevant sections of this manual. They are also listed below for reference:

  • Ctrl+O: Open a new file
  • Ctrl+S: Save the current project
  • Ctrl+Z: Undo
  • Ctrl+Y: Redo
  • Ctrl+Left: Previous image
  • Ctrl+Right: Next image

2.4.   Resizing

Images can be scaled down using the resize tool. Open the tool using the following button:

Use the slider in the resize tool to select the percent to resize by. New image dimensions are previewed under the "New Dimensions:" label:

Like the crop function above, the cancel button will exit the resize tool without making changes and the confirm button will commence the resize operation. The confirm button will be disabled if the slider is at 100% or if the new dimensions are less than 1 x 1.

NOTE: Resizing images typically results in some loss in quality.

2.5.   Brightness and Contrast

Image brightness and contrast can be adjusted by clicking this button:

This will display two sliders - the top slider adjusts the brightness and the bottom adjusts contrast. Brightness and contrast are increased or decreased by moving the sliders to the right or left, respectively.

The brightness and contrast adjustments are shown real-time in the image viewer. Similar to the crop and resize functions, the cancel button will cancel the operation and the confirm button will apply the changes.


3.   Managing Your Changes

3.1.   Undo

Changes to your project can undone by pressing Ctrl+Z or using the undo button in the upper left-hand corner:

The history of changes to your project is stored until the program is closed or a new image is opened. This button is disabled when there are no changes to undo. 'Save As' actions are not stored and cannot be undone.

3.2.   Redo

Undone changes can be redone with Ctrl+Y or the redo button:

This button is disabled when there are no changes to redo. The undo history is only stored immediately after changes are undone. If a new editing function is performed the undo history is cleared and the redo button is disabled.

3.3.   Save

Images can be saved using the save button next to the menu button or by pressing Ctrl+S:

This will save any changes made to the image. The button will be disabled if there are no changes to save. Once an image is edited, the button will be enabled and an asterisk(*) will be appended to the displayed name.

WARNING: This button will always overwrite the image file that is open in JIVE.

3.4.   Save As

Images can be saved as different names, in different directories, and as different file types using the save as feature:

On-click, JIVE will open a file explorer (Windows 10 shown below):

You can then navigate to the desired destination directory. File name can be edited by typing in the text box and file type can be selected using the file-type drop-down list.

Images can be converted to any of the compatible formats (JPG, PNG, BMP, or GIF).

The file type can be selected from the drop-down menu or appended to the file name. For example, entering the file name "image.jpg" will convert the image to a JPG and override the file type in the "Save As Type:" list. If no file extension is appended to the file name, JIVE will save the image as the file type selected in the "Save As Type:" drop-down menu.

The current image file will stay open in JIVE. The newly saved image will not be opened automatically.

NOTE: JPGs and BMPs do not support transparency. If you are converting from a PNG or GIF with transparency, the transparent pixels will be converted to white. Transparency is preserved when converting between PNG and GIF.
